# Liquid Glass Theme Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Upgrade the desktop app to React 19 and add a user-selectable `Liquid Glass` theme that applies `liquid-glass-react` only to high-value floating UI surfaces.

**Architecture:** Keep `minimal` as the default theme and add `glass` as an explicit Zustand theme value. Add a focused `GlassSurface` wrapper that renders plain semantic containers outside the glass theme and `LiquidGlass` inside the glass theme. Apply the wrapper only to the new-task dialog, send button wrapper, follow-up composer, and environment card host.

**Tech Stack:** Tauri v2, Vite, React 19, TypeScript, Zustand, Tailwind CSS, shadcn/Radix primitives, `liquid-glass-react`, Node contract tests.

### Task 4: Create `GlassSurface`

**Files:**
- Create: `apps/desktop/src/components/ui/GlassSurface.tsx`

- [ ] **Step 1: Create the wrapper component**

Create `apps/desktop/src/components/ui/GlassSurface.tsx` with this content:

```tsx
import LiquidGlass from 'liquid-glass-react';
import type { HTMLAttributes, ReactNode } from 'react';

import { cn } from '@/lib/utils';
import { useAppStore } from '@/state/appStore';

export type GlassSurfaceVariant = 'dialog' | 'composer' | 'panel' | 'button';

type GlassSurfaceElement = 'div' | 'section' | 'aside' | 'span';

interface GlassSurfaceProps extends HTMLAttributes<HTMLElement> {
  as?: GlassSurfaceElement;
  children: ReactNode;
  variant: GlassSurfaceVariant;
}

const glassProps: Record<
  GlassSurfaceVariant,
  {
    aberrationIntensity: number;
    blurAmount: number;
    cornerRadius: number;
    displacementScale: number;
    elasticity: number;
    overLight: boolean;
    padding: string;
    saturation: number;
  }
> = {
  dialog: {
    aberrationIntensity: 1.4,
    blurAmount: 0.08,
    cornerRadius: 24,
    displacementScale: 44,
    elasticity: 0.16,
    overLight: true,
    padding: '0',
    saturation: 128,
  },
  composer: {
    aberrationIntensity: 1.6,
    blurAmount: 0.075,
    cornerRadius: 24,
    displacementScale: 52,
    elasticity: 0.22,
    overLight: true,
    padding: '0',
    saturation: 132,
  },
  panel: {
    aberrationIntensity: 1.2,
    blurAmount: 0.065,
    cornerRadius: 26,
    displacementScale: 34,
    elasticity: 0.12,
    overLight: true,
    padding: '0',
    saturation: 122,
  },
  button: {
    aberrationIntensity: 1.8,
    blurAmount: 0.05,
    cornerRadius: 999,
    displacementScale: 48,
    elasticity: 0.28,
    overLight: true,
    padding: '0',
    saturation: 136,
  },
};

export function GlassSurface({
  as: Element = 'div',
  children,
  className,
  variant,
  ...props
}: GlassSurfaceProps) {
  const theme = useAppStore((state) => state.theme);

  if (theme !== 'glass') {
    return (
      <Element className={className} {...props}>
        {children}
      </Element>
    );
  }

  return (
    <Element
      className={cn('glass-surface-host', `glass-surface-host-${variant}`, className)}
      {...props}
    >
      <LiquidGlass
        {...glassProps[variant]}
        className={cn('glass-surface', `glass-surface-${variant}`)}
      >
        <div className="glass-surface-inner">{children}</div>
      </LiquidGlass>
    </Element>
  );
}
```
