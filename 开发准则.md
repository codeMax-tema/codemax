## 使用GitHub作为项目仓库托管



**VS Code 插件清单**

| 插件 ID                                 | 名称                            | 是否必装 | 用途                                                         |
| --------------------------------------- | ------------------------------- | -------- | ------------------------------------------------------------ |
| `GitHub.vscode-pull-request-github`     | GitHub Pull Requests and Issues | 必装     | 在 VS Code 内登录 GitHub、查看/创建 PR、处理 Issue、做代码审查。 |
| `GitHub.vscode-github-actions`          | GitHub Actions                  | 必装     | 查看和调试 GitHub Actions 工作流，检查 CI 构建、测试、发布日志。 |
| `eamodio.gitlens`                       | GitLens                         | 必装     | 增强 Git 能力，查看提交历史、文件 blame、分支、作者和变更记录。 |
| `rust-lang.rust-analyzer`               | rust-analyzer                   | 必装     | Rust 语言服务，用于 Tauri 后端代码补全、跳转、诊断和重构。   |
| `tauri-apps.tauri-vscode`               | Tauri                           | 必装     | Tauri 项目支持，识别配置、提供 Tauri 命令和开发辅助能力。    |
| `ms-python.python`                      | Python                          | 必装     | Python Agent 服务开发、调试、解释器选择和测试支持。          |
| `bradlc.vscode-tailwindcss`             | Tailwind CSS IntelliSense       | 必装     | Tailwind 类名补全、语法提示、样式检查。                      |
| `esbenp.prettier-vscode`                | Prettier                        | 必装     | 前端代码格式化，统一 TypeScript、JSON、CSS、Markdown 风格。  |
| `dbaeumer.vscode-eslint`                | ESLint                          | 必装     | 前端代码质量检查，发现 TypeScript/React 常见问题。           |
| `EditorConfig.EditorConfig`             | EditorConfig                    | 建议安装 | 读取 `.editorconfig`，统一缩进、换行、编码等基础格式。       |
| `vadimcn.vscode-lldb`                   | CodeLLDB                        | 建议安装 | Rust/Tauri 调试器，用于断点调试 Rust 后端。                  |
| `charliermarsh.ruff`                    | Ruff                            | 建议安装 | Python lint/format，检查 Agent 代码风格和常见问题。          |
| `tamasfe.even-better-toml`              | Even Better TOML                | 建议安装 | `Cargo.toml`、`pyproject.toml` 等 TOML 文件高亮、校验和补全。 |
| `redhat.vscode-yaml`                    | YAML                            | 建议安装 | GitHub Actions workflow、配置文件的 YAML 校验和补全。        |
| `Gruntfuggly.todo-tree`                 | Todo Tree                       | 建议安装 | 汇总项目里的 TODO/FIXME/NOTE，方便追踪待办。                 |
| `streetsidesoftware.code-spell-checker` | Code Spell Checker              | 建议安装 | 检查英文拼写，减少变量名、文档和提交说明里的拼写错误。       |



## **一键安装命令**

在 VS Code 终端执行：

```
code --install-extension GitHub.vscode-pull-request-github
code --install-extension GitHub.vscode-github-actions
code --install-extension eamodio.gitlens
code --install-extension rust-lang.rust-analyzer
code --install-extension tauri-apps.tauri-vscode
code --install-extension ms-python.python
code --install-extension bradlc.vscode-tailwindcss
code --install-extension esbenp.prettier-vscode
code --install-extension dbaeumer.vscode-eslint
code --install-extension EditorConfig.EditorConfig
code --install-extension vadimcn.vscode-lldb
code --install-extension charliermarsh.ruff
code --install-extension tamasfe.even-better-toml
code --install-extension redhat.vscode-yaml
code --install-extension Gruntfuggly.todo-tree
code --install-extension streetsidesoftware.code-spell-checker
```

**备注**

- 必装插件用于 GitHub 协作、Tauri/Rust、React/TypeScript、Python Agent 和 Tailwind 开发。
- 建议安装插件用于调试、格式统一、配置文件校验和日常质量提升。
- 不建议随意安装来路不明的 Git/AI/代码上传类插件，项目会涉及本地路径、密钥和仓库权限
